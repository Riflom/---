# ArtiQ — веб-сайт для тренировки дикции и артикуляции

Говори чётко. Звучи уверенно. Будь услышанным.

## Для кого создан ArtiQ?

ArtiQ разработан для всех, кто хочет улучшить качество своей речи — независимо от профессии, возраста или уровня подготовки:

- Студенты — для защиты проектов, выступлений, онлайн-презентаций  
- Преподаватели, лекторы, репетиторы — чтобы быть понятными и выразительными  
- Актёры, ведущие, подкастеры — для отработки дикции и артикуляции  
- Офисные сотрудники и фрилансеры — для уверенных выступлений на Zoom, Teams и других платформах  
- Просто те, кто хочет говорить чище, громче и выразительнее — без «проглатывания» окончаний и запинок

## Зачем это нужно? В чём польза?

Чёткая речь — это не только про «красиво говорить». Это про:

- Понимание — вас слышат с первого раза  
- Доверие — чёткая речь вызывает уверенность у собеседников  
- Профессионализм — особенно важно при публичных выступлениях  
- Самоуважение — вы перестаёте стесняться своей речи  
- Эффективность — меньше недопонимания, быстрее доносите мысль

В условиях цифровой коммуникации (видеозвонки, лекции онлайн, голосовые сообщения) умение говорить ясно стало базовым навыком — таким же важным, как грамотность.

## Функционал сайта

ArtiQ — это полностью клиентское веб-приложение, не требующее регистрации, установки или подключения к серверу. Все функции работают прямо в браузере:

1. **Библиотека упражнений**  
   - Скороговорки (от простых к сложным)  
   - Артикуляционные упражнения  
   - Тексты для отработки интонации и темпа речи  
   - Три уровня сложности: Начальный, Средний, Продвинутый

2. **Интерактивный тренажёр**  
   - Кнопка «Следующее упражнение» — генерирует случайное задание  
   - Крупный, читаемый текст — удобно читать вслух  
   - Опциональное выделение слов (как в караоке) для ритма

3. **Работа с микрофоном**  
   - Запись речи — через встроенный микрофон (MediaRecorder API)  
   - Прослушивание — сразу после записи, без загрузки  
   - Распознавание речи — браузер «переводит» вашу речь в текст (Web Speech API), чтобы вы увидели, насколько чётко говорите

Важно: аудио и данные никогда не покидают ваш браузер. Никаких серверов, никакой передачи данных.

4. **Раздел «Зачем это нужно?»**  
   - Краткое, но содержательное объяснение пользы тренировки дикции  
   - Кнопка «Вернуться к тренировке» — мгновенный возврат к упражнениям

5. **Локальное сохранение прогресса**  
   - Пройденные упражнения отмечаются и сохраняются в памяти браузера (localStorage)  
   - Данные остаются только у вас и не сбрасываются при обновлении страницы

## Как пользоваться сайтом?

1. Откройте сайт в браузере (рекомендуется Google Chrome или Microsoft Edge — они лучше всего поддерживают Web Speech API)  
2. Нажмите «Разрешить доступ к микрофону», когда браузер запросит (это нужно только для записи и распознавания речи)  
3. Выберите уровень сложности (или оставьте случайный выбор)  
4. Читайте вслух предложенное упражнение  
5. Нажмите:  
   - «Записать» — чтобы послушать себя  
   - «Распознать» — чтобы увидеть, как браузер «понял» вашу речь  
6. Хотите узнать, почему это важно? — нажмите кнопку «Зачем тренировать дикцию?» в шапке сайта  
7. Вернитесь к тренировке в любой момент

Совет: тренируйтесь ежедневно по 5–10 минут — уже через неделю вы заметите разницу!

## Как запустить локально (для разработчика или проверки)

Если вы скачали исходный код проекта (например, с GitHub):

1. Распакуйте архив в любую папку на компьютере  
2. Откройте файл `index.html` в браузере (двойной клик)  
3. Готово! Сайт работает без установки сервера
